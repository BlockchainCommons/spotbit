Spotbit

Price info services have long been the biggest privacy hole in Bitcoin. Though Bitcoin Core can run using Tor, and though new wallets like the Gordian Wallet can communicate with Bitcoin Core through Tor, Bitcoin price services traditially have not. This creates a potential red flag in a state that is antagonistic toward Bitcoin.
Spotbit is the answer. It is a portable Flask API for Bitcoin price data and candles. It can either be used as a repository of historical data that allows for more frequent API requests, or as a simple wrapper around exchange APIs that permits the user to collect information over Tor. It can aggregate data from over 100 exchanges and serve them from a single URL or as onion hidden service using Tor. Spotbit is extremely flexible: the user can decide which base currencies to use (USDT, USD, EUR, etc.), which exchanges to keep data for, and how much data to keep.
