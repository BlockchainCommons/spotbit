This Onion v3 Address is Running Spotbit

What is Spotbit?

Spotbit is a decentralized alternative to services such as coinmarketcap and coingecko. Spotbit aggregates data for dozens of exchanges and provides their data over a single tor API endpoint.

How do I use Spotbit?

Spotbit is an API service running on this server. To access it, you can use the API endpoints it provides: /configure will give you information on how this server is configured. /now/[currency]/[exchange] will give you the current information about a certain currency. And /hist/[currency]/[exchange]/[start_date]/[end_date] gives historical data.

You can find out more about how to use Spotbit here

Example Usage

Fetching current data for a specific currency and exchange

Currency: USD, Exchange: Coinbasepro /now/USD/coinbasepro

Fetching current averaged data for a currency

Currency: USD. Spotbit takes the simple average of the 5 exchanges listed in the README for a currency. The data returned is an average open price, average high price, average low price, and average close price. /now/USD
